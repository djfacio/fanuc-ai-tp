# Local FANUC Manuals Manifest

Copy this template into the appropriate folder under
`C:\Dev\Fanuc Robot Manuals` and maintain it next to the manual files. Do not
put licensed PDFs in this public repository.

## Manuals Root

```text
C:\Dev\Fanuc Robot Manuals
```

## Entries

| File | Title | Document number | Revision | Notes |
| --- | --- | --- | --- | --- |
| `example.pdf` | Example title | B-00000EN | 00 | Scope or controller option notes |

## Reference Rules

- Cite title, document number, revision, and section when adding project notes.
- Prefer controller-specific manuals over generic internet references.
- For generated robot code, verify system variables and controller behavior
  against this manifest, the manual itself, or the actual controller.

# FANUC Manuals

Place local FANUC manuals, option manuals, controller documentation, and extension documentation in this folder.

This folder is intentionally ignored except for this README because FANUC manuals are usually licensed or copyrighted and should not be published with this public repository.

Suggested organization:

```text
docs/fanuc-manuals/
  controller/
  karel/
  pc-developer-kit/
  roboguide/
  snpx/
  options/
```

When referencing manuals from project notes, cite the manual title, revision, and section instead of copying large excerpts into the repo.
